# Resume Screening System

## Overview
This project is a Machine Learning and NLP based Resume Screening System.

## Features
- Resume text extraction
- Resume preprocessing
- Resume similarity scoring
- Candidate ranking
- Missing skills detection
- Resume analysis report

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLTK
- PDFPlumber

## Output
The system compares resumes with a job description, calculates a similarity score, ranks candidates, and identifies missing skills.
